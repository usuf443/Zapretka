## 🐛 ЕСЛИ ЧТО-ТО НЕ РАБОТАЕТ

### Сообщения не синхронизируются?

**Проверка 1:** WebSocket сервер запущен
```bash
npm start
```
Должны увидеть: `Server listening on ws://localhost:3000`

**Проверка 2:** HTTP сервер запущен
```bash
python -m http.server 8000
```
Должны увидеть: `Serving HTTP on ...`

**Проверка 3:** Откройте консоль браузера (F12)
- Нет красных ошибок
- Должны быть сообщения типа: `Connected to WebSocket`

**Проверка 4:** Обновите страницу (F5)
- Сообщения должны загрузиться из localStorage

### Не можете войти?

1. **Пароль верный?** Пароль для всех: `1234`
2. **Имя правильное?** Без пробелов в конце
   - ✅ Саня
   - ❌ Саня  (с пробелом)
3. **Браузер поддерживает?** Используйте Chrome, Firefox, Safari, Edge

### Сообщения пропадают после обновления?

1. localStorage отключен в браузере?
   - Откройте настройки браузера
   - Разрешите локальное хранилище
2. localStorage переполнен?
   - Откройте F12 → Application → Local Storage
   - Удалите старые данные (начинающиеся с `zapretka_`)
