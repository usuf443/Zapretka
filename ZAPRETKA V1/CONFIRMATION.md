## 💬 Синхронизация между браузерами ✅

**Технология:** WebSocket + BroadcastChannel + localStorage

**Как это работает:**
1. Браузер 1 (Саня) отправляет сообщение
2. Сообщение уходит на WebSocket сервер (ws://localhost:3000)
3. WebSocket сервер передаёт сообщение всем подключённым браузерам
4. Браузер 2 (Маша) получает сообщение в реальном времени
5. Сообщение отображается в чате

**Файл:** `script.js`
```javascript
// WebSocket клиент
const ws = new WebSocket('ws://localhost:3000');
ws.onmessage = (e) => {
    const data = JSON.parse(e.data);
    if (data.type === 'msg') {
        messages.push(data.msg);
        renderMessages();  // Обновляем чат
    }
};

// При отправке сообщения
function doSend() {
    const msg = {
        id: Date.now() + '-' + Math.random().toString(36).substr(2, 5),
        from: me.name,
        to: activeContact,
        text: input.value,
        time: new Date().toLocaleTimeString(...),
        type: 'text'
    };
    
    // Отправляем на сервер
    ws.send(JSON.stringify({ type: 'msg', msg }));
}
```

## 🔄 СИНХРОНИЗАЦИЯ: 3 уровня

### Уровень 1️⃣: WebSocket (ПЕРВЫЙ ВЫБОР)
- Кросс-браузерная синхронизация
- Сервер: ws://localhost:3000
- Идеален для Chrome + Firefox

### Уровень 2️⃣: BroadcastChannel (FALLBACK)
- Синхронизация между вкладками одного браузера
- Если WebSocket недоступен
- 2 вкладки 1 браузера будут синхронизированы

### Уровень 3️⃣: localStorage (ULTIMATE FALLBACK)
- Однопользовательское хранилище
- Если всё остальное не работает
- Сообщения сохраняются локально
