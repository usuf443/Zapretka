Запуск локального WebSocket сервера и тестирование межбраузерных сообщений

1) Установите Node.js (если ещё нет).
2) В папке проекта (где находится `server.js` и `package.json`) выполните в PowerShell:

```powershell
npm install
npm start
```

3) Запустите локальный HTTP сервер для фронтенда (в отдельном терминале):

```powershell
python -m http.server 8000
```

4) Откройте в браузере две вкладки: http://localhost:8000
   - В каждой вкладке введите разное имя (например, "Артем" и "Евгений").
   - Перейдите в чат с другим пользователем и отправляйте сообщения — они будут передаваться через WebSocket сервер и появляться в обеих вкладках.

Примечания:
- Если WebSocket-соединение недоступно, клиент использует локальную синхронизацию через BroadcastChannel/localStorage (только для вкладок в одном браузере).
- Для реального использования между разными машинами убедитесь, что сервер доступен по сети (настройте firewall/порт forwarding или разверните на VPS).
