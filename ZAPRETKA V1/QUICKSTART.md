```bash
python -m http.server 8000

# Пишите сообщения друг другу 💬
```

### Вариант 2: Кросс-браузерная синхронизация (Chrome + Firefox)
```bash
# Терминал 1: запустите WebSocket сервер
npm install
npm start

# Терминал 2: запустите HTTP-сервер
python -m http.server 8000

# Chrome: http://localhost:8000 → Войти как "Саня"
# Firefox: http://localhost:8000 → Войти как "Маша"
# Пишите сообщения между браузерами 🔄
```

## 🎯 Что попробовать

1. **Поиск** 🔍
   - Кликните на 🔍 поиск в сайдбаре
   - Начните печатать имя пользователя или текст сообщения
   - Результаты показывают совпадающие контакты и сообщения

2. **Прокрутка вниз** ↓
   - Прокрутите чат вверх
   - Появится красная кнопка ↓ внизу справа
   - Кликните, чтобы перейти к последним сообщениям

3. **Профиль** 👤
   - Кликните на свой аватар в левом верхнем углу
   - Отредактируйте имя, псевдоним, био
   - Смените аватар (загрузите картинку или используйте gravatar)
   - Нажмите **Выход**, чтобы выйти из аккаунта

4. **Отправка файлов** 📎
   - Нажмите скрепку 📎 рядом с полем ввода
   - Выберите изображение
   - Отправляется вместе с сообщением

5. **Голосовые сообщения** 🎙️
   - Нажмите 🎙️ микрофон
   - Говорите
   - Нажмите ещё раз для отправки

6. **Эмодзи** 😊
   - Нажмите 😊 эмодзи кнопку
   - Выберите эмодзи из панели
   - Добавится в поле ввода

## 📋 Требования

- Node.js 12+ (если использовать WebSocket сервер)
- Python 3+ (для HTTP-сервера)
- Любой современный браузер (Chrome, Firefox, Safari, Edge)

## 🐛 Если что-то не работает

1. **Сообщения не синхронизируются между браузерами?**
   - Убедитесь, что `npm start` запущен (WebSocket сервер на :3000)
   - Проверьте консоль браузера (F12) на ошибки

2. **Кнопка поиска не работает?**
   - Обновите страницу (F5)
   - Убедитесь, что JavaScript включен

3. **Не можете войти?**
   - Пароль для всех: **1234**
   - Имя не чувствительно к регистру (Саня, саня, САНЯ — все работают)

4. **Сообщения пропадают после обновления страницы?**
   - Убедитесь, что localStorage не отключен в браузере
   - Проверьте, что localStorage имеет достаточно места (обычно 5-10MB)

## 📞 Поддержка

Если возникли проблемы:

2. Посмотрите на ошибки
3. Проверьте, что все файлы загружены правильно
