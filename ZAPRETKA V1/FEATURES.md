## 💬 Мессенджинг
- ✅ **Отправка сообщений** — в реальном времени между браузерами (WebSocket)
- ✅ **Прокрутка вверх/вниз** — свободная навигация по чату без принудительного скролла
- ✅ **Кнопка "Вниз"** ↓ — появляется когда вы не внизу чата, кликните чтобы перейти вниз
- ✅ **Persistent-сообщения** — все сообщения сохраняются в localStorage
- ✅ **Incremental рендеринг** — новые сообщения добавляются без перезагрузки всего чата

## 🔍 Поиск
- ✅ **Поиск по контактам** — введите имя в поле поиска в сайдбаре
- ✅ **Поиск по сообщениям** — найдите сообщения по содержимому и автору
- ✅ **Группировка результатов** — результаты отсортированы по чатам
- ✅ **Переход в чат** — клик по результату откроет выбранный чат

## 👤 Профиль
- ✅ **Редактирование профиля** — имя, псевдоним, место рождения, год, био
- ✅ **Смена аватара** — загрузите изображение или используйте gravatar
- ✅ **Сохранение данных** — профиль синхронизируется между вкладками/браузерами
- ✅ **Статус "Online"** — индикатор онлайна рядом с именем

## 📱 Интерфейс
- ✅ **Telegram-style дизайн** — трёхколончная верстка (сайдбар, чат, стикеры)
- ✅ **Glassmorphism** — полупрозрачные панели с blur эффектом
- ✅ **Sticky input** — поле ввода всегда видно внизу, не прячется под кнопками
- ✅ **Dark theme** — тёмная тема с цветами (розовый #ff4d6d, голубой #2a9fff)
- ✅ **Emoji panel** — 20+ популярных эмодзи на кнопку в чате
- ✅ **Responsive** — адаптивен для разных размеров экрана

## 📎 Медиа
- ✅ **Голосовые сообщения** — запись с микрофона (кнопка 🎙️)
- ✅ **Отправка файлов/изображений** — кнопка скрепки для загрузки
- ✅ **Превью файлов** — отображение изображений в чате

## 🤖 Zapret Bot
- ✅ **AI-like ответы** — бот автоматически отвечает на сообщения
- ✅ **Контекстные реплики** — разные ответы на разные сообщения
- ✅ **Реалистичная задержка** — бот отвечает с небольшой задержкой

## 🔄 Синхронизация
- ✅ **WebSocket** — кросс-браузерная синхронизация в реальном времени
- ✅ **BroadcastChannel** — синхронизация между вкладками одного браузера
- ✅ **localStorage fallback** — если WebSocket недоступен, сообщения хранятся локально
- ✅ **Архив чатов** — архивирование с синхронизацией через BroadcastChannel

## 🎮 Дополнительно
- ✅ **Бургер меню** — заглушка для расширенного меню
- ✅ **Модальные окна** — для профиля, звонков, настроек
- ✅ **Звонки (демо)** — модальное окно с кнопками принять/отклонить

## 🚀 Как запустить

### 1️⃣ Запустить WebSocket сервер (если нужна кросс-браузерная синхронизация)
```bash
npm install
npm start
```

### 2️⃣ Запустить HTTP сервер (в отдельном терминале)
```bash
python -m http.server 8000
```

### 3️⃣ Открыть в браузере
```
http://localhost:8000
```

### 4️⃣ Протестировать
- **Один браузер**: откройте 2 вкладки, войдите разными пользователями
- **Два браузера**: откройте http://localhost:8000 в двух браузерах, войдите разными пользователями
- **Отправляйте сообщения** — они синхронизируются в реальном времени
