### 1️⃣ script.js - Функция `renderMessages()`

**ДО:**
```javascript
if (!userIsScrolling && wasNearBottom) box.scrollTop = box.scrollHeight;
```


## 🎁 Бонус: Как отключить автоскролл (если потребуется)

Если когда-нибудь нужно отключить автоскролл, просто измените в `script.js`:

```javascript
// Из этого:
setTimeout(() => { box.scrollTop = box.scrollHeight; }, 10);

// На это:
// setTimeout(() => { box.scrollTop = box.scrollHeight; }, 10);  // комментарий
```

Или условие:
```javascript
// Скроллим только если пользователь внизу
const atBottom = (box.scrollHeight - box.scrollTop - box.clientHeight) < 100;
if (atBottom) {
    setTimeout(() => { box.scrollTop = box.scrollHeight; }, 10);
}
```
