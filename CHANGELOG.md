## 🔄 Логика поиска

1. Пользователь вводит текст в поле поиска
2. В реальном времени ищутся:
   - **Контакты** по имени (из массива `users`)
   - **Сообщения** по содержимому и автору (из массива `messages`)
3. Результаты группируются и отображаются в сайдбаре
4. Клик на результат:
   - Переходит в выбранный чат
   - Очищает поле поиска
   - Показывает сообщения из этого чата

## 🚀 Как использовать

### Запуск
```bash
# WebSocket сервер (для кросс-браузерной синхронизации)
npm install && npm start

# HTTP сервер (в отдельном терминале)
python -m http.server 8000

# Открыть http://localhost:8000
```

### Тестирование
1. Откройте 2 браузера или 2 вкладки
2. Войдите как разные пользователи (пароль `1234`)
3. Пишите сообщения друг другу
4. Используйте поиск: кликните на 🔍 и введите текст
