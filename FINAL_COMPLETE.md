## 🎓 КАК ЭТО РАБОТАЕТ

### Автоскролл вниз:
```
1. Пользователь отправляет сообщение
2. Сообщение добавляется в DOM
3. Функция renderMessages() вызывается
4. setTimeout(10ms) позволяет браузеру отрисовать
5. box.scrollTop = box.scrollHeight ← скролл вниз
6. Пользователь видит новое сообщение внизу ✅
```

### Ручная прокрутка:
```
1. Пользователь крутит мышкой/трекпадом
2. CSS overflow-y: auto ← позволяет скроллить
3. scroll-behavior: smooth ← плавное движение
4. Пользователь видит старые/новые сообщения ✅
```
