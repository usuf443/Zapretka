## 🚀 3 КОМАНДЫ ДО РАБОТЫ

### Команда 1: Установка
```bash
npm install
```

### Команда 2: WebSocket сервер (Терминал 1)
```bash
npm start
```
✅ Увидите: `Server listening on ws://localhost:3000`

### Команда 3: HTTP сервер (Терминал 2)
```bash
python -m http.server 8000
```
✅ Увидите: `Serving HTTP on 0.0.0.0 port 8000`
